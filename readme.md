## Install Instructions

- Clone and run `npm install`

- Configure Homestead yaml file
- Configure Hosts file
- Run vagrant machine
- Modify `.env.example` to be `.env` and add db config
- Run `npm run dev`
- Run `php artisan migrate`
- Should be good to go
